About This Application

This is a Flask-based web application that integrates with the Wikipedia API to provide users with a simple way to search for and view summaries of Wikipedia articles.

Features

Search for any topic using the search functionality

View concise summaries of Wikipedia articles

Access direct links to full Wikipedia pages for more details

About the Project

This application is built as part of a practical exercise to demonstrate web development with Flask, API integration, and template-based UI design.

This Application is based on a demo project by Lindsay Ward, with additional modifications by Guanyu Lu.
